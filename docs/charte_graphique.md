# Charte Graphique (Doc2Form)

## Ton Visuel et Identité
- **Vibe** : Technologique, propre, confiant, focalisé sur la productivité. Un aspect documentaire et studieux croisé avec de la technologie d'IA moderne.

## Typographie
- **Primaire (UI, Textes longs)** : Inter (sans-serif)
- **Secondaire (Titres, Accents)** : Space Grotesk pour un aspect technique/digital et moderne.
- **Monospace (Code, Statut)** : JetBrains Mono

## Couleurs
- **Couleurs Principales** :
  - Primary : `#6366F1` (Indigo 500) - Pour les boutons primaires et la distinction de marque.
  - Background : `#FAFAFA` (Zinc 50) pour un mode clair très lisible et studieux.
  - Surface : `#FFFFFF` (Banc pur) pour les zones de dépose (Dropzone) et les cartes (Cards).
- **Couleurs Sémantiques** :
  - Succès : `#10B981` (Emerald 500)
  - Attention/Upload : `#F59E0B` (Amber 500)
  - Erreur : `#EF4444` (Red 500)
- **Textes** :
  - Principal : `#18181B` (Zinc 900)
  - Secondaire : `#71717A` (Zinc 500)

## UI et Composants
- **Dropzone** : Grandes bordures pointillées (`border-dashed`), avec indication visuelle claire lors du glisser-déposer. Transition douce sur le hover.
- **Cartes de questions** : Élégantes, ombres subtiles (`shadow-sm`, `shadow-md` au survol), bordures grises légères (`border-zinc-200`), coins arrondis (`rounded-xl`).
- **Boutons** : Solides et clairs, effet de ripple ou transition sur `hover:opacity-90`, coins arrondis (`rounded-lg`).

## Iconographie
- Utilisée via *Lucide React*.
- Icônes de type 'Document', 'UploadCloud', 'FileText', 'Image', 'CheckCircle'. Lignes fines, taille modérée (16-24px).
